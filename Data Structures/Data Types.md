# Data Types

## Integers

>> + Uses 4 bytes of memory.
>> + Range is  => 0 to 2^32-1 for unsigned and -2^31 to 2^31-1 for signed integers.

## Floats 

>> + Size = 4 bytes(upto 7 decimal digits)
>> + Double => Size = 8 bytes(upto 15 decimal digits)

## Characters

>> + Size = 1 byte
>> + ASCII Table is used

## Boolean

>> + Size = 1 byte
>> + True => 1 ; False => 0
